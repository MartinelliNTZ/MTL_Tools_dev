      - Ensures errors are captured in plugin JSONL logs and QGIS message log when critical
  1.7.1.7
    - Security: avoid Bandit B310 by replacing `urllib.request.urlopen` with `http.client` requests
      - `core/task/altimetry_task.py`: use `http.client.HTTPSConnection`/`HTTPConnection` with validated scheme
      - `core/task/reverse_geocoding_task.py`: same replacement and validation
      - Adds explicit `User-Agent`, checks HTTP status and returns descriptive errors
  1.7.1.8
    - ReverseGeocode: follow HTTP 3xx redirects (max 5)

      - Log redirect locations and errors via `LogUtils`
  1.7.1.9
    - Fix restart subprocess security (avoid shell=True)
      - `plugins/RestartQgis.py`: do not create .bat or call subprocess with `shell=True`
